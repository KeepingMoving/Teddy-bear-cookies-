## 🌙 关于故事 (About The Story)

**每个人都有觉得自己要“碎掉”的时候。**

这款游戏源于一个关于孤独与自我接纳的温柔隐喻。玩家将扮演一块在社交场合中感到格格不入、最终“碎掉”的小熊饼干。

这不仅仅是一个游戏，更是一次**心理疗愈的旅程**。

我们利用**“金缮” (Kintsugi)** 的艺术理念——用黄金修补破碎的器物——来传达一个核心思想：**那些曾经伤害你的裂痕，终将变成你身上最闪耀的金线。**

## ✨ 游戏体验 (Gameplay Experience)

一段从黑暗走向黎明的沉浸式互动体验：

1.  **社交的窘迫 (The Social Anxiety)**
    *   体验身处喧闹人群中的孤独感，面对尴尬的社交抉择，直到不堪重负而“破碎”。

2.  **迷雾中的寻找 (The Search in Darkness)**
    *   跌入内心的黑暗迷雾（Darkness Phase）。
    *   化作一束微光，避开消极的情绪，寻找那些代表“温柔”、“勇气”和“本心”的灵魂碎片。
    *   收集萤火虫带来的暖心话语。

3.  **金缮修复 (Kintsugi Repair)**
    *   这是一个拼图过程，也是重塑自我的过程。
    *   将散落的身体部位（耳朵、圆脸、温暖的手）重新拼凑。
    *   感受碎片归位时那一瞬间的治愈与安宁。

4.  **黎明与拥抱 (Sunrise & Embrace)**
    *   当一切修复完成，黑夜退去，温暖的日出降临。
    *   **互动彩蛋**：点击重生的小熊，摸摸它的头，给它一个拥抱，它会回应你的爱意。

## 🎨 设计美学 (Design Aesthetics)

*   **极简治愈风**：使用柔和的琥珀色（Amber）与深邃的石板蓝（Slate）作为主色调，模拟从深夜到日出的光影变化。
*   **纯代码绘图**：游戏中的小熊、碎片、光影效果均由 SVG 和 CSS 代码实时渲染，没有使用一张贴图，以此获得最丝滑的交互体验。
*   **动态叙事**：打字机效果的独白，配合呼吸般的动画节奏，让故事缓缓流淌进心里。

## 🛠️ 技术栈 (Tech Stack)

本项目使用现代前端技术构建：

*   **Core**: React 18 (Hooks, Custom Components)
*   **Language**: TypeScript (Strict typing for game states)
*   **Styling**: Tailwind CSS (Animation, Layout, Responsive Design)
*   **Icons**: Lucide React
*   **Animation**: CSS Keyframes & Transitions
